# Pixiv Headless

Stable pixiv crawler middleware for headless.

## Start

```bash
git clone https://github.com/ArsFy/pixiv-headless.git
cd pixiv-headless
npm install
```

> Puppeteer: `node node_modules/puppeteer/install.js`

#### Set Token

`./token`: random text

#### HTTP API

```
POST /api/get_image
```

| Key | Description |
| --- | ----------- |
| i  | Pixiv ID    |
| token | `./token` random text |
